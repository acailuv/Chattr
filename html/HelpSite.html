Chattr! is a socket.io powered chatting application.
Please visit this site for more information about Chattr!.

Chattr! is a chatting application that is capable to:
1. Send and recieve messages or chatting with other user in a room.
2. Transfer files with other user.
3. Create a room for public and private purposes.
4. Private chatting with another user.
5.

Setting Up...

1. You will need nodejs and XAMPP to be installed in your computer. Download the latest version of nodejs here and XAMPP here.
2. After you have downloaded and installed nodejs and XAMPP, clone this repository (or download) and place it somewhere nice.
3. Set your XAMPP that's located in your local disk and start both Apache and MySQL.
4. Open that Chattr! folder, hold shift and right click on an empty space.
5. Click the "Open PowerShell window here". A Windows PowerShell terminal will pop up in that directory.
6. Type in the following command node ./index.js and hit enter.
7. Wait until there is an output saying: listening on *:3000.
8. Done. The server is up and running.

Connecting to Server...

1. Open your preferred browser.
2. Type in localhost:3000 in the url text bar and hit enter.
3. Done. You are connected to Chattr!. Feel free to explore this simple application.
